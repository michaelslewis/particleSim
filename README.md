# Particle Simulation with OpenCL and OpenGL

A GPU-accelerated particle simulation built with OpenCL and OpenGL, demonstrating parallel computation, graphics programming, and real-time visualization.

The simulation models a large number of particles interacting with geometric objects while leveraging OpenCL for parallel processing and OpenGL for rendering and visualization.

## Technologies

- C
- OpenCL
- OpenGL
- Parallel Computing
- GPU Programming
- Computer Graphics
